## 一、任务目标

实现消息汇总面板，显示所有 AI 角色的交互消息，每条消息有转发按钮。

---

## 二、功能需求

### UI 结构
```
┌──────────────────────────────────────────────────┐
│  消息汇总                                [过滤]   │
├──────────────────────────────────────────────────┤
│  [用户 → 项目经理 PM] 10:30                      │
│  实现用户登录功能                                 │
│  └─ [转发]                                       │
│                                                   │
│  [项目经理 PM → 用户] 10:31                      │
│  收到！我会安排前端工程师实现登录页面...          │
│  └─ [转发]                                       │
└──────────────────────────────────────────────────┘
```

### 核心功能
1. 显示所有交互消息（时间线视图）
2. 每条消息显示：发送者 → 接收者、时间、内容
3. 每条消息有"转发"按钮
4. 支持过滤（按角色、时间、关键词）
5. 只显示最终响应，不显示推理过程

---

## 三、实现要点

- 监听 `IAISessionManagerService.onDidMessageAppend` 事件
- 使用 `IMessageHubService` 存储和管理消息
- 消息格式：`{ from, to, content, timestamp }`
