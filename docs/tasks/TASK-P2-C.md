## 一、任务目标

实现消息转发机制，将消息转发给其他 AI 角色，目标角色自动被选中。

---

## 二、功能需求

### 转发流程
```
点击"转发"按钮 → 选择目标角色 → 确认转发 → 目标角色自动被选中
```

### 核心功能
1. 点击消息的"转发"按钮，弹出目标角色选择对话框
2. 选择目标角色，点击"确认转发"
3. 消息转发给目标角色
4. 目标角色自动被选中（团队列表中变蓝）
5. 聊天输入框的目标切换到该角色

---

## 三、实现要点

- 调用 `IAISessionManagerService.relayMessage()` 转发消息
- 转发后，调用 `setActiveSession()` 切换选中角色
- 更新团队列表的选中状态
