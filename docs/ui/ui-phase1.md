## 本阶段新增界面

Phase 1 预计新增/改造以下界面：

- **AI Team Commander Panel**（主控面板，全新）
- **角色创建向导**（新建角色的对话框/分步骤界面）
- **任务下达区域**（每个角色卡片内嵌的对话框）
- **消息中继操作**（中继按钮 + 目标选择弹窗）

## 基于界面的验收操作步骤

- [ ] 打开 AI Team Panel，验证角色卡片正常显示
- [ ] 点击"添加成员"，验证创建向导正确运行
- [ ] 选择角色模板，填写名称，完成创建，验证卡片出现在列表中
- [ ] 向某个角色发送任务，观察状态变为"工作中"（绿色动画）
- [ ] 任务完成后，观察状态恢复"空闲"
- [ ] 点击"中继"按钮，选择目标角色，确认，验证消息发送成功
- [ ] 切换不同角色，验证对话历史各自独立
