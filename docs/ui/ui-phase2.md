## 本阶段新增界面

Phase 2 预计新增/改造以下界面：

- **工作流编排画布**（节点拖拽 + 连线的核心编辑器）
- **节点库面板**（可用节点类型的选择列表）
- **工作流运行监控视图**（实时高亮当前执行节点）
- **任务进度总览**（整个自动化流程的执行状态）
- **上下文管理界面**（全局项目记忆的查看与编辑）

## 基于界面的验收操作步骤

- [ ] 打开工作流编辑器，验证画布可以拖拽平移
- [ ] 从节点库拖出角色节点，放置到画布
- [ ] 连接两个节点，验证连线正确建立
- [ ] 添加条件分支节点，配置分支条件
- [ ] 启动工作流，观察节点高亮流转
- [ ] 中途暂停工作流，验证暂停操作有效
- [ ] 查看上下文管理面板，验证项目摘要显示正确
